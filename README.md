# Norak App
Norak is an Islamic application built for the Pi Network ecosystem, providing users with a comprehensive spiritual experience.

## Features
- **Holy Quran:** Full Quran integration connected via an external library for high-quality recitation and text.
- **Athkar & Dua:** Daily prayers and remembrances for ease of access.
- **Pi Browser Compatible:** Fully optimized for seamless performance within the Pi Browser.

## Technical Details
This project is open-source under the **PiOS** (Pi Open Source) license, ensuring collaboration within the Pi Network community.
